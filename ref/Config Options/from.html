Sends and verifies confirmation emails

(Quick Reference)

from

This plugin configuration value is declared using Platform Core configuration conventions.

The value is used as the default From: address on confirmation e-mails is no from argument is specified when calling the emailConfirmationServiceMethods method.

Example

Here is how you would configure this in Config.groovy:

plugin.emailConfirmation.from = '"Do not reply" <noreply@mymegacorp.com>'
